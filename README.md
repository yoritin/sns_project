## URL
http://sns-project.yoriblog.com/

## 機能
- ログイン機能
- ユーザー登録機能
- ログアウト機能

- ユーザーページ
  - ユーザーアイコン登録、変更機能（縦横比を維持したまま、丸くなる）
  - 名前、メールアドレス、パスワード変更（テストユーザーは不可）

- 投稿機能(ログインユーザーのみ)
  - 投稿一覧表示
  - 投稿編集機能
  - 投稿削除
  - 投稿に対してコメント機能
  - いいね機能
  - いいね解除

- フォロー機能
  - ユーザーページからフォローできる
  - タイムライン機能（フォローしたユーザーの投稿が一覧表示出来る）
  - フォロー解除機能
