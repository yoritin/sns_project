## アイディアを共有するSNSサイトです。
URL 
## 機能
- ユーザー登録
- ログイン
- アイディア
  - 投稿、編集、削除
- アイディアにコメント
  - 投稿、削除
- フォロー
  - フォローする
  - フォロー解除
  - フォロー、フォロワー数カウント
- いいね機能
  - いいね登録
  - いいね解除
  - いいね数カウント
- ユーザーのプロフィール編集機能
  - 名前、メール、パスワード変更
  - ユーザーアイコン、自己紹介文登録
